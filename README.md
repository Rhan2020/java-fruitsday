# java-fruitsday
一个基于 Java、JSP、Servlet、MySQL的水果商城。

### 实现功能

* 登陆、注册
* 浏览所有商品
* 查看商品详情
* 添加购物车
* 添加关注
* 管理员功能
	* 用户的增、删、改、查
	* 商品的增、删、改、查
